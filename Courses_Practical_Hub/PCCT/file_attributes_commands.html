File Attribute Commands

attrib

Displays or changes file attributes. You can manage Read-only (R), Archive (A), System (S), and Hidden (H) attributes.

Syntax:

  • View attributes: attrib [filename]
  • Set an attribute: attrib +[attribute] [filename]
  • Remove an attribute: attrib -[attribute] [filename]

Example: Manage attributes for a file named report.txt.

1. Check current attributes:

C:\JIC\Docs>attrib report.txt
A                    C:\JIC\Docs\report.txt

2. Add the Read-only attribute:

C:\JIC\Docs>attrib +r report.txt

3. Check attributes again:

C:\JIC\Docs>attrib report.txt
A    R               C:\JIC\Docs\report.txt

4. Add Hidden and System attributes:

C:\JIC\Docs>attrib +h +s report.txt

5. Remove the Read-only attribute (Note: This may fail if System or Hidden is set):

C:\JIC\Docs>attrib -r report.txt
Not resetting system file - C:\JIC\Docs\report.txt

6. Remove System and Hidden first, then remove Read-only:

C:\JIC\Docs>attrib -s -h report.txt
C:\JIC\Docs>attrib -r report.txt
C:\JIC\Docs>attrib report.txt
A                    C:\JIC\Docs\report.txt
