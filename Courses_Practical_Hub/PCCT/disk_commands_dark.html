Disk Management Commands

format

Prepares a storage drive for use by creating a new file system. Warning: This erases all data on the drive.

C:\>format D: /fs:ntfs /q /v:Data
The type of the file system is FAT32.
New file system is NTFS.
QuickFormatting 953869M
Creating file system structures.
Format complete.
...

diskpart

A powerful command-line utility for advanced disk management. Type diskpart to enter its environment.

list disk / list volume

Displays available disks or volumes.

DISKPART> list disk

  Disk ###  Status         Size     Free     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
* Disk 0    Online          465 GB      0 B        *
  Disk 1    Online          931 GB   1024 KB       *

select disk / select volume

Sets the focus to a specific disk or volume.

DISKPART> select disk 1

Disk 1 is now the selected disk.

clean

Removes all partition and volume information from the selected disk. This erases all data.

DISKPART> clean

DiskPart succeeded in cleaning the disk.

create partition primary

Creates a new primary partition on the selected disk.

DISKPART> create partition primary size=500000

DiskPart succeeded in creating the specified partition.

active

Marks the selected partition as active, which means it is a bootable system partition.

DISKPART> active

DiskPart marked the current partition as active.

assign letter

Assigns a drive letter to the selected volume.

DISKPART> assign letter=E

DiskPart successfully assigned the drive letter or mount point.

filesystems

Displays the file systems supported for formatting.

DISKPART> filesystems

  File System  Type        Size
  -----------  ----------  -----------------------
  FAT          FAT         Up to 4 GB
  FAT32        FAT32       Up to 32 GB
  exFAT        exFAT       Up to 128 PB
  NTFS         NTFS        Up to 256 TB
