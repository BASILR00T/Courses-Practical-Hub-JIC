Python Basics

Introduction to Python

Python is a high-level, interpreted programming language known for its clear syntax and readability. It's a great first language for beginners and is widely used in web development, data science, automation, and more.

1. "Hello, World!" - Your First Program

The first step in any language is to print a simple message. In Python, this is done with the print() function.

print("Hello, World!")

Output:

Hello, World!

2. Variables and Data Types

Variables are used to store data. Python is dynamically typed, meaning you don't need to declare the data type.

# String (text)
name = "Basil"

# Integer (whole number)
age = 25

# Float (number with a decimal)
height = 175.5

# Boolean (True or False)
is_student = True

print("Name:", name)
print("Age:", age)
print("Is a student:", is_student)

Output:

Name: Basil
Age: 25
Is a student: True

3. Basic Operators

Python supports standard arithmetic and comparison operators.

# Arithmetic Operators
a = 10
b = 3
print("Addition:", a + b)       # 13
print("Subtraction:", a - b)    # 7
print("Multiplication:", a * b) # 30
print("Division:", a / b)         # 3.33...
print("Modulus (remainder):", a % b) # 1

# Comparison Operators
print("Is a > b?", a > b)        # True
print("Is a == 10?", a == 10)   # True

4. Input and Output

You can get input from the user with the input() function and display output with print().

user_name = input("Enter your name: ")
print("Hello, " + user_name + "!")

Example Interaction:

Enter your name: Ahmed
Hello, Ahmed!

5. Control Flow (if/elif/else)

Control the flow of your program based on conditions.

score = 85

if score >= 90:
    print("Grade: A")
elif score >= 80:
    print("Grade: B")
else:
    print("Grade: C or lower")

Output:

Grade: B

6. Loops

Repeat blocks of code using for and while loops.

For Loop

Iterates over a sequence (like a list or a range of numbers).

# Loop 5 times
for i in range(5):
    print("Loop number", i)

Output:

Loop number 0
Loop number 1
Loop number 2
Loop number 3
Loop number 4

While Loop

Repeats as long as a condition is true.

count = 0
while count < 3:
    print("Count is", count)
    count = count + 1

Output:

Count is 0
Count is 1
Count is 2

7. Functions

Functions are reusable blocks of code. You define them with the def keyword.

# Define a function
def greet(name):
    return "Hello, " + name + "!"

# Call the function
message = greet("Sarah")
print(message)

Output:

Hello, Sarah!

8. Basic Data Structures

Lists

An ordered, mutable collection of items.

fruits = ["apple", "banana", "cherry"]
print(fruits[0])  # Access the first item
fruits.append("orange") # Add a new item
print(fruits)

Output:

apple
['apple', 'banana', 'cherry', 'orange']

Dictionaries

A collection of key-value pairs.

student = {
    "name": "Faisal",
    "id": 439101,
    "major": "COIT"
}
print(student["name"]) # Access value by key
print(student)

Output:

Faisal
{'name': 'Faisal', 'id': 439101, 'major': 'COIT'}
