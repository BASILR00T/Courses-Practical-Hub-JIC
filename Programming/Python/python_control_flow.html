Python Control Flow

Conditional Statements (if, elif, else)

تُستخدم الجمل الشرطية لتنفيذ كود معين بناءً على تحقق شرط ما.

temperature = 28

if temperature > 30:
    print("It's a hot day.")
elif temperature > 20: # (30 >= temperature > 20)
    print("It's a nice day.")
else:
    print("It's cold.")
It's a nice day.

Loops (for and while)

حلقات التكرار تُستخدم لتنفيذ كتلة من الكود عدة مرات.

For Loop

# التكرار على قائمة
fruits = ["apple", "banana", "cherry"]
for fruit in fruits:
    print(fruit)

# التكرار لعدد معين من المرات
for i in range(5): # from 0 to 4
    print(i)

While Loop

count = 0
while count < 5:
    print(count)
    count += 1 # نفس الشيء مثل count = count + 1
