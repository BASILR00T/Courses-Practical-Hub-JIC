Python Data Structures

Lists

القوائم هي مجموعات مرتبة وقابلة للتغيير من العناصر.

numbers = [1, 2, 3, 4, 5]
# إضافة عنصر في النهاية
numbers.append(6)
print(f"After append: {numbers}")

# الوصول إلى عنصر عن طريق الفهرس (يبدأ من 0)
print(f"First element: {numbers[0]}")

# حذف عنصر عن طريق الفهرس
numbers.pop(2) # حذف العنصر في الفهرس 2 (وهو الرقم 3)
print(f"After pop: {numbers}")
After append: [1, 2, 3, 4, 5, 6]
First element: 1
After pop: [1, 2, 4, 5, 6]

Dictionaries

القواميس هي مجموعات غير مرتبة من أزواج (مفتاح: قيمة). كل مفتاح يجب أن يكون فريدًا.

person = {"name": "Fatima", "age": 22, "city": "Dammam"}

# الوصول إلى قيمة عن طريق المفتاح
print(f"Name: {person['name']}")

# إضافة زوج جديد
person["major"] = "IT"
print(f"After adding major: {person}")

# حذف زوج
del person["age"]
print(f"After deleting age: {person}")
Name: Fatima
After adding major: {'name': 'Fatima', 'age': 22, 'city': 'Dammam', 'major': 'IT'}
After deleting age: {'name': 'Fatima', 'city': 'Dammam', 'major': 'IT'}

Tuples and Sets

Tuples

الـ Tuples هي مجموعات مرتبة ولكنها **غير قابلة للتغيير** بعد إنشائها.

coordinates = (10.0, 20.0)
print(f"X: {coordinates[0]}, Y: {coordinates[1]}")

# السطر التالي سيتسبب في خطأ لأن الـ Tuples غير قابلة للتغيير
# coordinates[0] = 15  # TypeError

Sets

الـ Sets هي مجموعات غير مرتبة ولا تسمح بالعناصر المكررة.

numbers = {1, 2, 3, 4, 4, 4}
print(f"Set of numbers: {numbers}")

numbers.add(5)
print(f"After adding 5: {numbers}")

numbers.remove(2)
print(f"After removing 2: {numbers}")
Set of numbers: {1, 2, 3, 4}
After adding 5: {1, 2, 3, 4, 5}
After removing 2: {1, 3, 4, 5}
