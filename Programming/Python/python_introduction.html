Introduction to Python

Variables and Data Types

المتغيرات هي حاويات لتخزين قيم البيانات. بايثون لا تحتاج إلى تعريف نوع المتغير بشكل صريح.

# String (نص)
name = "Basil"
# Integer (عدد صحيح)
age = 25
# Float (عدد عشري)
height = 175.5
# Boolean (قيمة منطقية)
is_student = True

print(f"Name: {name}, Type: {type(name)}")
print(f"Age: {age}, Type: {type(age)}")
Name: Basil, Type: <class 'str'>
Age: 25, Type: <class 'int'>

String Methods

النصوص في بايثون تأتي مع العديد من الدوال المدمجة (methods) لمعالجتها.

message = "  Hello, World!  "

# تحويل النص إلى حروف كبيرة
print(message.upper())

# إزالة المسافات البيضاء من البداية والنهاية
print(message.strip())

# استبدال جزء من النص
print(message.replace("World", "Python"))

# تقسيم النص إلى قائمة
print(message.strip().split(','))
  HELLO, WORLD!  
Hello, World!
  Hello, Python!  
['Hello', ' World!']

Formatted Strings (f-strings)

f-strings هي طريقة سهلة ومقروءة لتضمين المتغيرات داخل النصوص.

name = "Ali"
age = 30
print(f"My name is {name} and I am {age} years old.")
My name is Ali and I am 30 years old.
