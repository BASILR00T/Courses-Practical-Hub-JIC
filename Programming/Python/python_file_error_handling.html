File & Error Handling

Error Handling (try...except)

تُستخدم لمعالجة الأخطاء التي قد تحدث أثناء تشغيل البرنامج دون أن يتوقف. هذا يمنع تعطل البرنامج ويسمح لك بتقديم رسائل خطأ مفيدة للمستخدم.

try:
    number = int(input("Enter a number: "))
    result = 10 / number
    print(f"Result is {result}")
except ValueError:
    print("Invalid input. Please enter a number.")
except ZeroDivisionError:
    print("You cannot divide by zero.")
except Exception as e:
    print(f"An unexpected error occurred: {e}")
إذا أدخل المستخدم "abc":
Invalid input. Please enter a number.
إذا أدخل المستخدم "0":
You cannot divide by zero.

File I/O (Input/Output)

كيفية قراءة البيانات من الملفات النصية والكتابة إليها. استخدام `with open(...)` يضمن إغلاق الملف تلقائيًا.

Writing to a File

# 'w' للكتابة (سوف ينشئ الملف إذا لم يكن موجودًا أو يمسح محتواه إذا كان موجودًا)
with open("greeting.txt", "w") as file:
    file.write("Hello from Python!\n")
    file.write("This is a new line.")

Reading from a File

# 'r' للقراءة
with open("greeting.txt", "r") as file:
    content = file.read()
    print(content)
Hello from Python!
This is a new line.

Appending to a File

# 'a' للإضافة إلى نهاية الملف دون مسح المحتوى القديم
with open("greeting.txt", "a") as file:
    file.write("\nAppending another line.")
