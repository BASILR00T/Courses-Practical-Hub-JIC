NAT Configuration

Static NAT

Static NAT maps a single private IP address to a single public IP address. It is typically used for servers that need to be accessible from the outside.

  1. Define the inside interface.
  2. Define the outside interface.
  3. Configure the static mapping between the inside local address and the inside global address. 
! Define the inside interface
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip nat inside
Router(config-if)# exit

! Define the outside interface
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip nat outside
Router(config-if)# exit

! Configure Static NAT mapping
Router(config)# ip nat inside source static 192.168.1.10 209.165.200.225

Dynamic NAT

Dynamic NAT maps a private IP address to a public IP address from a pool of available public IP addresses.

  1. Define the inside and outside interfaces.
  2. Create a pool of public IP addresses.
  3. Create an Access Control List (ACL) to define which private addresses can be translated.
  4. Bind the ACL to the NAT pool.
! Define interfaces (same as above)
! Create a pool of public addresses
Router(config)# ip nat pool MY_POOL 209.165.200.226 209.165.200.240 netmask 255.255.255.224

! Create ACL for private addresses
Router(config)# access-list 1 permit 192.168.1.0 0.0.0.255

! Bind the ACL to the Pool
Router(config)# ip nat inside source list 1 pool MY_POOL

PAT (NAT Overload)

Port Address Translation (PAT) is the most common form of NAT. It translates multiple private IP addresses into a single public IP address by using different port numbers to track connections.

  1. Define the inside interface (your local network).
  2. Define the outside interface (facing the internet/public network).
  3. Create an Access Control List (ACL) to identify the private IP addresses that are allowed to be translated.
  4. Enable NAT, linking the ACL to the outside interface and enabling overload.
! Define the inside interface
Router(config)# interface GigabitEthernet0/0
Router(config-if)# ip nat inside
Router(config-if)# exit

! Define the outside interface
Router(config)# interface GigabitEthernet0/1
Router(config-if)# ip nat outside
Router(config-if)# exit

! Create an access list to permit your private network range
Router(config)# access-list 1 permit 10.0.0.0 0.255.255.255

! Enable NAT, linking the access list to the outside interface with overload
Router(config)# ip nat inside source list 1 interface GigabitEthernet0/1 overload
