Basic Cisco IOS CLI

1. Navigating CLI Modes

Cisco IOS has a hierarchical structure. You move through different modes to perform different tasks. The prompt changes to show your current mode.

  • User EXEC Mode (Switch>): For basic monitoring and viewing commands.
  • Privileged EXEC Mode (Switch#): Also called "enable mode". Provides access to all monitoring commands as well as device management commands like `reload` and `copy`.
  • Global Configuration Mode (Switch(config)#): The gateway to all configuration changes.
  • Specific Configuration Modes (e.g., Switch(config-if)#): For configuring specific parts of the device, like an interface or a routing protocol.
! From User EXEC, enter Privileged EXEC mode
Switch>enable
Switch#

! From Privileged EXEC, enter Global Configuration mode
Switch#configure terminal
Enter configuration commands, one per line.  End with CNTL/Z.
Switch(config)#

! Exit back one level
Switch(config)#exit
Switch#

! Return directly to Privileged EXEC mode from any config mode
Switch(config-if)#end
Switch#

2. First Steps & Identification

These are some of the first commands you should execute on a new device to give it a unique identity.

Hostname

Changes the name of the switch or router, which is reflected in the CLI prompt.

Switch(config)#hostname SW1
SW1(config)#

Banner MOTD (Message of the Day)

Sets a banner that is displayed to anyone who connects to the device. It's often used for legal warnings.

! The '#' is a delimiting character; the banner ends when it sees the second '#'
SW1(config)#banner motd # UNAUTHORIZED ACCESS IS PROHIBITED #

3. Basic Interface Configuration

Interfaces are the physical ports on the device. They must be configured to pass traffic.

! Enter interface configuration mode for GigabitEthernet 0/1
SW1(config)#interface gigabitEthernet 0/1

! Add a description to the interface (highly recommended)
SW1(config-if)#description Link to Core Router

! On a router or Layer 3 switch, assign an IP address
SW1(config-if)#ip address 192.168.1.1 255.255.255.0

! Enable the interface (ports are disabled by default)
SW1(config-if)#no shutdown
SW1(config-if)#
%LINK-5-CHANGED: Interface GigabitEthernet0/1, changed state to up
%LINEPROTO-5-UPDOWN: Line protocol on Interface GigabitEthernet0/1, changed state to up

! To disable an interface
SW1(config-if)#shutdown

4. Essential 'Show' Commands

`show` commands are used in User EXEC or Privileged EXEC mode to view the device's status and configuration. They are essential for verification and troubleshooting.

! Display the current, active configuration
SW1#show running-config

! Display the configuration that will be loaded on reboot
SW1#show startup-config

! A summary of all interfaces, their IP addresses, and status (very useful!)
SW1#show ip interface brief
Interface              IP-Address      OK? Method Status                Protocol
GigabitEthernet0/1     192.168.1.1     YES manual up                    up
GigabitEthernet0/2     unassigned      YES manual administratively down down

! Display detailed information about the IOS version and hardware
SW1#show version
Cisco IOS Software, C2960 Software (C2960-LANBASEK9-M), Version 15.0(2)SE4...
...

5. Saving and Erasing the Configuration

Changes made in global configuration mode are applied immediately to the `running-config`. To make them permanent, you must save them to the `startup-config`.

Saving the Configuration

! Copies the active configuration from RAM to NVRAM
SW1#copy running-config startup-config
Destination filename [startup-config]?
Building configuration...
[OK]

Erasing the Configuration (Factory Reset)

To completely wipe a switch and return it to its factory default state, you must erase the startup configuration and the VLAN database, then reload.

! Erase the startup configuration file
SW1#erase startup-config
Erasing the nvram filesystem will remove all configuration files! Continue? [confirm]
[OK]
Erase of nvram: complete

! Delete the VLAN database file
SW1#delete flash:vlan.dat
Delete filename [vlan.dat]?
Delete flash:/vlan.dat? [confirm]

! Reboot the switch
SW1#reload
Proceed with reload? [confirm]

6. Using CLI Help

The `?` is your most powerful tool. Use it to see available commands or the options for a command you're typing.

! See all commands that start with 'cl'
SW1#cl?
clear  clock

! See the next required parameter for 'clock set'
SW1#clock set ?
  hh:mm:ss  Current Time

! See what comes next
SW1#clock set 14:30:00 ?
  <1-31>  Day of the month
  MONTH   Month of the year

! Keep going until you see  (carriage return)
SW1#clock set 14:30:00 1 October 2025 ?
