Switch Port Security

Configuring Port Security

Port security is a feature that restricts input to an interface by limiting and identifying MAC addresses of the stations allowed to access the port. This prevents unauthorized users from connecting to the network.

  1. Set the port to access mode.
  2. Enable port security on the interface.
  3. (Optional) Set the maximum number of allowed MAC addresses. Default is 1.
  4. Define the violation action (shutdown, restrict, or protect). shutdown is the most secure and default action.
  5. Set the method for learning MAC addresses. sticky learns dynamically and saves the address to the running configuration.
Switch(config)# interface FastEthernet0/1
Switch(config-if)# switchport mode access
Switch(config-if)# switchport port-security
Switch(config-if)# switchport port-security maximum 1
Switch(config-if)# switchport port-security violation shutdown
Switch(config-if)# switchport port-security mac-address sticky

Verification

You can verify the port security configuration and see which MAC addresses have been learned.

! Show the overall port security status for all interfaces
Switch# show port-security
Secure Port  MaxSecureAddr  CurrentAddr  SecurityViolation  Security Action
                (Count)       (Count)          (Count)
---------------------------------------------------------------------------
      Fa0/1              1            1                  0         Shutdown
---------------------------------------------------------------------------
Total Addresses in System (configured and dynamic): 1

! Show detailed port security status for a specific interface
Switch# show port-security interface fastEthernet 0/1
Port Security              : Enabled
Port Status                : Secure-up
Violation Mode             : Shutdown

Total MAC Addresses        : 1

Security Violation Count   : 0

When a violation occurs (e.g., an unauthorized device connects), the port will enter an err-disabled state. To re-enable it, you must go to the interface configuration and issue the shutdown command followed by the no shutdown command.

Protect a Cisco switch from another unauthorized switch

Enable BPDU Guard (Prevents Rogue Switches on Access Ports)

Switch(config)#interface fastEthernet 0/1  (or gigabitEthernet as needed)
Switch(config-if)#spanning-tree portfast                    
Switch(config-if)#spanning-tree bpduguard enable
