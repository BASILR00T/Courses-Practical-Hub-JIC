Static & Default Routing

Static Routing

A static route is a manually configured entry in a router's routing table. It tells the router how to reach a specific network that is not directly connected.

The command syntax is: ip route [destination_network] [subnet_mask] [next_hop_ip_address]. 

! On Router1, to reach the 20.0.0.0/8 network, the next hop is 192.168.1.2
Router1(config)# ip route 20.0.0.0 255.0.0.0 192.168.1.2

! On Router2, to reach the 10.0.0.0/8 network, the next hop is 192.168.1.1
Router2(config)# ip route 10.0.0.0 255.0.0.0 192.168.1.1

Default Routing

A default route, also known as the "gateway of last resort", tells the router where to send traffic when it doesn't have a specific route for the destination network. This is essential for connecting to the internet.

A default route uses 0.0.0.0 0.0.0.0 to match all possible networks.

! On a router connected to an ISP, configure a default route to send all unknown traffic to the ISP's router
HQ_Router(config)# ip route 0.0.0.0 0.0.0.0 203.0.113.1
! 203.0.113.1 is the IP address of the next-hop ISP router

Verification

Router# show ip route
