Inter-VLAN Routing

Router on a Stick (ROAS)

This method uses a router with a single physical interface connected to a switch via a trunk link to route between multiple VLANs. The router's physical interface is divided into logical "subinterfaces", one for each VLAN.

! On the Switch, configure the port connected to the router as a trunk
Switch(config)# interface GigabitEthernet0/1
Switch(config-if)# switchport mode trunk
Switch(config-if)# exit

! On the Router, enable the physical interface
Router(config)# interface GigabitEthernet0/1
Router(config-if)# no shutdown
Router(config-if)# exit

! Create a subinterface for VLAN 10
Router(config)# interface GigabitEthernet0/1.10
! Set the encapsulation type to dot1q and specify the VLAN ID
Router(config-subif)# encapsulation dot1q 10
! Assign an IP address to be the default gateway for that VLAN
Router(config-subif)# ip address 192.168.10.1 255.255.255.0
Router(config-subif)# exit

! Create a subinterface for VLAN 20
Router(config)# interface GigabitEthernet0/1.20
Router(config-subif)# encapsulation dot1q 20
Router(config-subif)# ip address 192.168.20.1 255.255.255.0

Layer 3 Switch (SVI)

A Layer 3 (or multilayer) switch can perform routing functions, eliminating the need for an external router. This is done by creating Switched Virtual Interfaces (SVIs) for each VLAN and enabling IP routing.

! Enable IP routing on the Layer 3 switch
L3_Switch(config)# ip routing

! Create an SVI for VLAN 10 and assign its default gateway IP
L3_Switch(config)# Vlan 10
L3_Switch(config)# interface Vlan 10
L3_Switch(config-if)# ip address 10.0.0.1 255.0.0.0
L3_Switch(config-if)# no shutdown
L3_Switch(config-if)# exit

! Create an SVI for VLAN 20 and assign its default gateway IP
L3_Switch(config)# Vlan 20
L3_Switch(config)# interface Vlan 20
L3_Switch(config-if)# ip address 11.0.0.1 255.0.0.0
L3_Switch(config-if)# no shutdown
