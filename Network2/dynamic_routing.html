Dynamic Routing (RIP & OSPF)

RIP (Routing Information Protocol)

RIP is a distance-vector routing protocol that uses hop count as its metric (Max 15 hops; 16 is unreachable). It has an Administrative Distance (AD) of 120. It's simple to configure but is best suited for smaller networks.

! Enter RIP configuration mode
Router(config)# router rip
! Specify RIP version 2 (which is classless)
Router(config-router)# version 2
! Disable automatic summarization (Essential for discontiguous networks)
Router(config-router)# no auto-summary
! Advertise the directly connected networks
Router(config-router)# network 192.168.1.0
Router(config-router)# network 10.0.0.0

OSPF (Open Shortest Path First)

OSPF is a link-state routing protocol that is more scalable and efficient than RIP, making it suitable for larger networks. It uses cost as its metric, which is based on bandwidth.

! Enter OSPF configuration mode with a process ID (e.g., 1)
Router(config)# router ospf 1
! Advertise networks using a network address, wildcard mask, and area ID
Router(config-router)# network 192.168.1.0 0.0.0.255 area 0
Router(config-router)# network 10.10.10.0 0.0.0.255 area 0

The wildcard mask is the inverse of a subnet mask (e.g., for 255.255.255.0, the wildcard is 0.0.0.255). All routers in an OSPF setup must have at least one interface in area 0, the backbone area.

Verification

! Show the IP routing table to see routes learned via RIP (R) or OSPF (O)
Router# show ip route
...
O     192.168.2.0/24 [110/2] via 10.10.10.2, 00:05:32, Serial0/0/0
R     172.16.0.0/16 [120/1] via 10.10.10.2, 00:00:15, Serial0/0/0
...


! Show information about OSPF neighbors
Router# show ip ospf neighbor

Neighbor ID     Pri   State           Dead Time   Address         Interface
192.168.2.1       1   FULL/BDR        00:00:38    10.10.10.2      Serial0/0/0

After assigning IP addresses on each router interface, you need to perform the following steps to configure EIGRP routing:

On Router1

Router1(config)# router eigrp 13
Router1(config-router)# network 10.0.0.0
Router1(config-router)# network 100.0.0.0
Router1(config-router)# exit

On Router2

Router2(config)# router eigrp 13
Router2(config-router)# network 100.0.0.0
Router2(config-router)# network 192.168.1.0
Router2(config-router)# network 200.0.0.0
Router2(config-router)# exit

On Router3

Router3(config)# router eigrp 13
Router3(config-router)# network 200.0.0.0
Router3(config-router)# network 172.16.0.0
Router3(config-router)# exit

Verification
