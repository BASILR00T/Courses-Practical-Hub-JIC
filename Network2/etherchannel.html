EtherChannel (Link Aggregation)

Layer 2 EtherChannel

EtherChannel bundles multiple physical switch ports into a single logical link, providing higher bandwidth and redundancy. Configuration can be done using a dynamic protocol (LACP or PAgP) or manually.

LACP (Link Aggregation Control Protocol) - IEEE Standard

Uses active mode to initiate negotiation and passive mode to wait for the other side.

! On Switch1 (initiator)
SW1(config)# interface range GigabitEthernet0/1 - 2
SW1(config-if-range)# channel-group 1 mode active

! On Switch2 (listener)
SW2(config)# interface range GigabitEthernet0/1 - 2
SW2(config-if-range)# channel-group 1 mode passive

PAgP (Port Aggregation Protocol) - Cisco Proprietary

Uses desirable mode to initiate negotiation and auto mode to wait for the other side.

! On Switch1 (initiator)
SW1(config)# interface range GigabitEthernet0/1 - 2
SW1(config-if-range)# channel-group 1 mode desirable

! On Switch2 (listener)
SW2(config)# interface range GigabitEthernet0/1 - 2
SW2(config-if-range)# channel-group 1 mode auto

Layer 3 EtherChannel

A Layer 3 EtherChannel acts as a routed interface, not a switched one. This is useful for creating high-bandwidth routed links between multilayer switches.

  1. Convert the physical ports to Layer 3 interfaces using the no switchport command.
  2. Bundle them into a channel-group.
  3. Assign an IP address to the logical Port-channel interface.
Switch1(config)# interface range FastEthernet0/1 - 2
Switch1(config-if-range)# no switchport
Switch1(config-if-range)# channel-group 12 mode on
Switch1(config-if-range)# exit

Switch1(config)# interface Port-channel 12
Switch1(config-if)# ip address 192.168.12.1 255.255.255.0
Switch1(config-if)# no shutdown

Load balancing

After you create a an etherchannel you enable Load balancing on both switches by doing the following:

Note: If a question asked you for a specific type select it.

Switch1(config)# port-channel load-balance ?

 dst-ip       Dst IP Addr
 dst-mac      Dst Mac Addr
 src-dst-ip   Src XOR Dst IP Addr
 src-dst-mac  Src XOR Dst Mac Addr
 src-ip       Src IP Addr
 src-mac      Src Mac Addr
 
Switch(config)#port-channel load-balance dst-mac
