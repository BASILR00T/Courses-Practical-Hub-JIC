Services & Processes

Task Manager (taskmgr.exe)

Task Manager provides a real-time view of your system's performance and the applications and processes running on it. It's essential for troubleshooting performance issues and unresponsive applications.

To open Task Manager: Press Ctrl + Shift + Esc, or run taskmgr.exe.

Key Tabs & Functions:

  • Processes: View running applications and background processes. You can right-click an item and select "End task" to force it to close.
  • Performance: Monitor CPU, Memory, Disk, and Network usage with real-time graphs.
  • Details: A more advanced view of all running processes with their Process ID (PID), memory usage, and CPU time.

Services (services.msc)

Windows Services are programs that run in the background, often starting when the computer boots. They perform core OS functions. Misconfigured or failed services can cause system-wide problems.

To open Services: Press Windows Logo + R, type services.msc, and press Enter.

Managing a Service:

Find the service in the list and double-click it to open its properties. From here you can:

  • Start, Stop, Pause, or Restart the service.
  • Change its Startup type:
    • Automatic: Starts when Windows boots.
    • Automatic (Delayed Start): Starts shortly after boot to improve startup time.
    • Manual: Starts only when needed by an application or another service.
    • Disabled: Prevents the service from starting at all.
