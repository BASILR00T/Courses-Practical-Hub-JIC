User Management Commands

net user

Adds, modifies, and removes user accounts from an elevated Command Prompt.

Common Commands:

  • View all users: net user
  • Add a user: net user [username] [password] /add
  • Delete a user: net user [username] /delete
  • Change a password: net user [username] *

'net user' Commands

This command is used to add, display, modify, and delete user accounts. It must be run from an elevated Command Prompt.

Display All Users

C:\WINDOWS\system32>net user

User accounts for \\DESKTOP-JIC01
-------------------------------------------------------------------------------
Administrator            DefaultAccount           Guest
JIC_Student              WDAGUtilityAccount
The command completed successfully.

Add a New User

! Add a user with a password
C:\WINDOWS\system32>net user Ahmed 12345 /add
The command completed successfully.

! Add a user, prompting for the password interactively (to hide it)
C:\WINDOWS\system32>net user Ali * /add
Type a password for the user: *****
Retype the password to confirm: *****
The command completed successfully.

Important Note: If the username contains spaces, it must be enclosed in quotation marks.

C:\WINDOWS\system32>net user "Ahmed Admin" 123 /add
The command completed successfully.

Display Specific User Information

C:\WINDOWS\system32>net user Ahmed
User name                    Ahmed
Full Name
Comment
User's comment
Country/region code          000 (System Default)
Account active               Yes
Account expires              Never
...
Local Group Memberships      *Users
Global Group memberships     *None
The command completed successfully.

Activate or Deactivate a User

! Deactivate the user account
C:\WINDOWS\system32>net user Ahmed /active:no
The command completed successfully.

! Activate the user account
C:\WINDOWS\system32>net user Ahmed /active:yes
The command completed successfully.

Delete a User

C:\WINDOWS\system32>net user Ahmed /delete
The command completed successfully.

'net localgroup' Commands

This command is used to manage local groups on the computer.

Display All Groups

C:\WINDOWS\system32>net localgroup

Aliases for \\DESKTOP-JIC01
----------------------------------------------------------------
*Administrators
*Backup Operators
*Guests
*Power Users
*Users
The command completed successfully.

Add a New Group

C:\WINDOWS\system32>net localgroup Sales /add
The command completed successfully.

Add a User to a Group

! Add user "Ahmed" to the "Sales" group
C:\WINDOWS\system32>net localgroup Sales Ahmed /add
The command completed successfully.

! Add a user to the "Power Users" group (requires quotes)
C:\WINDOWS\system32>net localgroup "Power Users" Ahmed /add
The command completed successfully.

Other Useful Commands

hostname

Displays the current computer name on the network.

C:\>hostname
DESKTOP-JIC01

whoami

Displays the currently logged-on user.

C:\>whoami
desktop-jic01\jic_student

! Display the user's Security Identifier (SID)
C:\>whoami /user
USER INFORMATION
----------------
User Name         SID
================= =============================================
desktop-jic01\jic_student S-1-5-21-123456789-987654321-1234567890-1001

shutdown

Used to shut down or restart the computer.

! Shut down immediately and force running applications to close
C:\>shutdown /s /f /t 0

GUI Tool: lusrmgr.msc

For visual user and group management, you can use the "Local Users and Groups" tool.

C:\>lusrmgr.msc
[A graphical window opens showing Users and Groups]

From inside this tool, you can right-click on a user to perform tasks like "Set Password", "Properties" (to change group membership), or disable the account.
