User Management Commands

net user

Adds, modifies, and removes user accounts. This must be run from an elevated Command Prompt.

Syntax:

  • View all users: net user
  • Add a user: net user [username] [password] /add
  • Delete a user: net user [username] /delete
  • Change a password: net user [username] *

Example: Create, manage, and delete a user named TestUser.

1. Create a new user with a password:

C:\WINDOWS\system32>net user TestUser Pa$$w0rd /add
The command completed successfully.

2. View all user accounts to confirm creation:

C:\WINDOWS\system32>net user

User accounts for \\DESKTOP-JIC01
-------------------------------------------------------------------------------
Administrator            DefaultAccount           Guest
Student-Admin            TestUser                 WDAGUtilityAccount
The command completed successfully.

3. Change the user's password (you will be prompted):

C:\WINDOWS\system32>net user TestUser *
Type a password for the user: 
Retype the password to confirm: 
The command completed successfully.

4. Delete the user account:

C:\WINDOWS\system32>net user TestUser /delete
The command completed successfully.
