System Information Commands

systeminfo

Provides a detailed summary of the computer's hardware and software configuration.

Syntax: systeminfo

Example: Display system information in the console.

C:\Users\Student>systeminfo

Host Name:                 DESKTOP-JIC01
OS Name:                   Microsoft Windows 10 Enterprise
OS Version:                10.0.19041 N/A Build 19041
OS Manufacturer:           Microsoft Corporation
System Manufacturer:       HP
System Model:              EliteBook 840 G5
System Type:               x64-based PC
Processor(s):              1 Processor(s) Installed.
                           [01]: Intel64 Family 6 Model 142 Stepping 10 GenuineIntel ~2100 Mhz
BIOS Version:              HP Q78 Ver. 01.09.01, 1/25/2022
Total Physical Memory:     8,079 MB
Available Physical Memory: 3,456 MB
...

msinfo32

Opens the System Information graphical utility, which provides comprehensive technical details about the system.

Syntax: msinfo32

Example: Launch the System Information window.

C:\Users\Student>msinfo32
[A graphical window opens showing detailed system information]

dxdiag

Launches the DirectX Diagnostic Tool, used to troubleshoot video and sound card problems.

Syntax: dxdiag

Example: Open the DirectX Diagnostic Tool.

C:\Users\Student>dxdiag
[A graphical window opens with tabs for System, Display, Sound, and Input]

winver

Displays the "About Windows" dialog box, showing the specific Windows version and build number.

Syntax: winver

Example: Check the exact version of Windows.

C:\Users\Student>winver
[A dialog box appears with "Microsoft Windows", Version, and OS Build information]

dir

Lists files and directories. It can also be used to search for files with specific extensions.

Example: Find all Control Panel applets (.cpl files) in the System32 directory.

C:\WINDOWS\system32>dir *.cpl
