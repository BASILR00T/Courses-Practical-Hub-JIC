PC Config & Troubleshooting

A quick reference for practical CLI commands and tools covered in the JIC PC Configuration & Troubleshooting course.

System Information

Commands to view OS, hardware, and version details (systeminfo, msinfo32).

File Attributes

Manage file properties like read-only, hidden, and system using 'attrib'.

Disk Management

Using format, diskpart, and other advanced disk utilities.

Drive Maintenance

Check for errors and defragment drives with chkdsk and defrag.

User Management

Manage local accounts with 'net user' and 'lusrmgr.msc'.

System Recovery

Boot and system file repair commands like sfc, bootrec, and dism.

Device Management

Managing hardware and drivers with the 'devmgmt.msc' tool.

System Configuration

Using 'sysdm.cpl' for virtual memory and system restore points.

Services & Processes

Managing background tasks and applications with 'services.msc' and 'taskmgr.exe'.

Registry & Event Viewer

Using 'regedit.exe' and 'eventvwr.msc' for advanced troubleshooting.
