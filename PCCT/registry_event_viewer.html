Registry Editor & Event Viewer

Registry Editor (regedit.exe)

The Windows Registry is a hierarchical database that stores low-level settings for the operating system and for applications. Editing it can be dangerous, so it's crucial to back up before making changes.

To open Registry Editor: Press Windows Logo + R, type regedit.exe, and press Enter. (Requires administrator privileges).

Key Tasks:

  • Navigation: The left pane contains keys (folders). Navigate through them to find the setting you need to modify.
  • Editing Values: The right pane contains values (settings). Double-click a value to edit it.
  • Export (Backup): Before editing, right-click the key you plan to change and select "Export". This saves a .reg file that can be used to restore the original settings by double-clicking it.

Event Viewer (eventvwr.msc)

Event Viewer is a logbook for your computer. It records significant events, such as application crashes, security audits, driver failures, and system startups. It's one of the first places to look when troubleshooting a problem.

To open Event Viewer: Press Windows Logo + R, type eventvwr.msc, and press Enter.

Key Logs:

Under "Windows Logs" in the left pane, you'll find the most important logs:

  • Application: Events logged by applications or programs.
  • Security: Events related to security, such as login attempts (successful and failed).
  • Setup: Events related primarily to application setup.
  • System: Events logged by Windows system components, such as a driver failing to start. This is a critical log for troubleshooting hardware and OS issues.

Events are classified by level: Information, Warning, Error, and Critical. When troubleshooting, focus on the Error and Critical events first.
