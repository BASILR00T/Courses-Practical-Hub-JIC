System Recovery Commands

These commands are typically run from the Windows Recovery Environment (WinRE) Command Prompt to troubleshoot and repair boot and system file issues.

Accessing Windows RE

If Windows can still boot, you can force it to restart into the Recovery Environment with the following command:

C:\>shutdown /r /o
[A message appears that you're about to be signed out]

File & Boot Integrity Commands

sfc (System File Checker)

Scans and repairs corrupted or missing protected system files.

X:\Sources>sfc /scannow

Beginning system scan. This process will take some time.
Beginning verification phase of system scan.
Verification 100% complete.
Windows Resource Protection did not find any integrity violations.

dism (Deployment Image Servicing and Management)

Repairs the underlying component store that sfc uses for repairs.

X:\Sources>dism /online /cleanup-image /restorehealth

Deployment Image Servicing and Management tool
Image Version: 10.0.19042.867

[==========================100.0%==========================]
The restore operation completed successfully.

bootrec

Repairs boot-related issues like a corrupt Master Boot Record (MBR) or Boot Configuration Data (BCD).

  • bootrec /fixmbr - Writes a new MBR.
  • bootrec /fixboot - Writes a new boot sector.
  • bootrec /rebuildbcd - Scans for Windows installations and adds them to the boot list.
X:\Sources>bootrec /rebuildbcd

Scanning all disks for Windows installations.
Total identified Windows installations: 1
[1] C:\Windows
Add installation to boot list? Yes(Y)/No(N)/All(A): Y
The operation completed successfully.

Diagnostic & Repair Tools

chkdsk

Checks a disk for file system errors and bad sectors and attempts to repair them.

X:\Sources>chkdsk c: /r
The type of the file system is NTFS.
Chkdsk cannot run because the volume is in use by another process.
Chkdsk may run if this volume is dismounted first.
ALL OPENED HANDLES TO THIS VOLUME WOULD THEN BE INVALID.
Would you like to force a dismount on this volume? (Y/N) Y
Volume dismounted. All opened handles to this volume are now invalid.
...
Windows has scanned the file system and found no problems.

rstrui.exe

Launches the System Restore graphical utility, allowing you to revert to a previous restore point.

X:\Sources>rstrui.exe
[The System Restore graphical window opens]

mdsched.exe

Launches the Windows Memory Diagnostic tool to check your RAM for errors.

X:\Sources>mdsched.exe
[The Windows Memory Diagnostic graphical window opens]
