System Recovery Commands

These commands are typically run from the Windows Recovery Environment (WinRE) Command Prompt.

sfc

System File Checker. Scans the integrity of all protected system files and repairs files with problems when possible.

Syntax: sfc /scannow

X:\Sources>sfc /scannow

Beginning system scan. This process will take some time.

Beginning verification phase of system scan.
Verification 100% complete.

Windows Resource Protection did not find any integrity violations.

bootrec

Used to repair boot-related issues like a corrupt Master Boot Record (MBR) or Boot Configuration Data (BCD).

Syntax:

  • bootrec /fixmbr - Writes a new MBR to the system partition.
  • bootrec /fixboot - Writes a new boot sector.
  • bootrec /rebuildbcd - Scans all disks for Windows installations and prompts you to add them to the BCD.

Example: Rebuild the Boot Configuration Data.

X:\Sources>bootrec /rebuildbcd

Scanning all disks for Windows installations.

Please wait, since this may take a while...

Successfully scanned Windows installations.
Total identified Windows installations: 1
[1]  C:\Windows
Add installation to boot list? Yes(Y)/No(N)/All(A): Y
The operation completed successfully.

dism

Deployment Image Servicing and Management. Can be used to repair the Windows component store, which `sfc` uses as a source for repairs.

Syntax: dism /online /cleanup-image /restorehealth

C:\WINDOWS\system32>dism /online /cleanup-image /restorehealth

Version: 10.0.19041.844

Image Version: 10.0.19042.867
