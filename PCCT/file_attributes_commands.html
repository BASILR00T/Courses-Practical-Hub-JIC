File & Directory Commands

Basic File & Directory Commands

mkdir (md)

Creates a new directory (folder).

C:\Users\Jargon\Desktop>mkdir MIT_Project

echo

Creates a new text file and writes content to it using the output redirection symbol (>).

C:\Users\Jargon\Desktop\MIT_Project>echo This is a test file. > readme.txt

C:\Users\Jargon\Desktop\MIT_Project>type readme.txt
This is a test file.

attrib

Displays or changes file and folder attributes. You can manage Read-only (R), Archive (A), System (S), and Hidden (H) attributes.

Managing Attributes for a Single File

  • View attributes: attrib [filename]
  • Set an attribute: attrib +[attribute] [filename] (e.g., +h for hidden)
  • Remove an attribute: attrib -[attribute] [filename] (e.g., -r for read-only)

Example: Make a file read-only and hidden.

C:\MIT_Project>attrib +r +h readme.txt

C:\MIT_Project>attrib readme.txt
A    R H      C:\MIT_Project\readme.txt

Applying Attributes to a Folder and its Contents

You can use switches to apply attributes recursively to all files and subfolders.

  • /s : Processes matching files in the current folder and all subfolders.
  • /d : Processes folders as well.

Example: Make the entire MIT_Project folder, and everything inside it, a hidden system folder.

C:\Users\Jargon\Desktop>attrib +s +h MIT_Project/*.* /s /d

C:\Users\Jargon\Desktop>attrib MIT_Project
A    S H      C:\Users\Jargon\Desktop\MIT_Project

Important Note: If a file or folder has the System (S) or Hidden (H) attribute set, you may not be able to change other attributes until you remove them first.
