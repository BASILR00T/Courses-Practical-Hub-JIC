Disk Management Commands

Standard CMD Commands

These commands can be run directly from the Command Prompt without entering the Diskpart utility.

vol

Displays the volume label and serial number for a disk.

C:\>vol C:
 Volume in drive C is Windows
 Volume Serial Number is 1A2B-3C4D

label

Creates, changes, or deletes the volume label (name) of a disk.

C:\>label D: Backup

format

Prepares a storage drive for use by creating a new file system. Warning: This erases all data on the drive.

C:\>format D: /fs:ntfs /q /v:Data
The type of the file system is FAT32.
New file system is NTFS.
QuickFormatting 953869M...
Format complete.

Diskpart Utility

A powerful utility for advanced disk management. Start it by typing diskpart in an elevated Command Prompt.

Information Commands

Partition & Volume Management

// Select a volume to manage
DISKPART> select volume 3
Volume 3 is now the selected volume.

// Increase the size by 2GB (2000MB)
DISKPART> extend size=2000
DiskPart successfully extended the volume.

// Reduce the size by 5GB (5000MB)
DISKPART> shrink desired=5000
DiskPart successfully shrunk the volume by: 5000 MB

// Remove the drive letter
DISKPART> remove letter=E
DiskPart successfully removed the drive letter or mount point.

// Delete the selected partition or volume (ERASES ALL DATA)
DISKPART> delete volume
DiskPart successfully deleted the selected volume.

Virtual Hard Disk (VHD) Management

This combined example shows the complete workflow for creating, partitioning, and managing a new virtual disk.

// 1. Create a new 5GB virtual disk file
DISKPART> create vdisk file="d:\vms\coit.vhd" maximum=5000
  100 percent completed
DiskPart successfully created the virtual disk file.

// 2. Select and attach the new virtual disk
DISKPART> select vdisk file="d:\vms\coit.vhd"
DiskPart successfully selected the virtual disk file.
DISKPART> attach vdisk
  100 percent completed
DiskPart successfully attached the virtual disk file.

// 3. Clean the disk and convert to MBR style for this example
DISKPART> clean
DiskPart succeeded in cleaning the disk.
DISKPART> convert mbr
DiskPart successfully converted the selected disk to MBR format.

// 4. Create partitions (e.g., 2GB primary, 2GB extended, then logicals)
DISKPART> create partition primary size=2000
DiskPart succeeded in creating the specified partition.
DISKPART> create partition extended size=2000
DiskPart succeeded in creating the specified partition.
DISKPART> create partition logical size=1000
DiskPart succeeded in creating the specified partition.
DISKPART> create partition logical
DiskPart succeeded in creating the specified partition.

// 5. Format and assign letters (select each partition before formatting)
DISKPART> select partition 1
Partition 1 is now the selected partition.
DISKPART> format fs=ntfs quick label=Google
  100 percent completed
DiskPart successfully formatted the volume.
DISKPART> assign letter=G
DiskPart successfully assigned the drive letter or mount point.
// (Repeat for other partitions...)

// 6. Expand the virtual disk's maximum size (e.g., to 8GB)
DISKPART> expand vdisk maximum=8000
  100 percent completed
DiskPart successfully expanded the virtual disk.

// 7. When finished, detach the virtual disk
DISKPART> detach vdisk
DiskPart successfully detached the virtual disk file.

Creating a Bootable USB Drive

Use Diskpart to prepare a USB flash drive to be a bootable Windows installation medium. Warning: This will erase all data on the USB drive.

// 1. Identify your USB drive (e.g., Disk 2)
DISKPART> list disk
  Disk ###  Status         Size     Free     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
  Disk 0    Online          465 GB      0 B        *
  Disk 1    Online          931 GB      0 B        *
  Disk 2    Online           58 GB      0 B

// 2. Select the USB drive
DISKPART> select disk 2
Disk 2 is now the selected disk.

// 3. Wipe the drive
DISKPART> clean
DiskPart succeeded in cleaning the disk.

// 4. Create a primary partition
DISKPART> create partition primary
DiskPart succeeded in creating the specified partition.

// 5. Select and mark the partition as active (bootable)
DISKPART> select partition 1
Partition 1 is now the selected partition.
DISKPART> active
DiskPart marked the current partition as active.

// 6. Format the partition as FAT32
DISKPART> format fs=fat32 quick label="WinSetup"
  100 percent completed
DiskPart successfully formatted the volume.

// 7. Assign a drive letter
DISKPART> assign letter=W
DiskPart successfully assigned the drive letter or mount point.

// 8. Exit diskpart
DISKPART> exit
Leaving DiskPart...

Warning: After completing these steps, you must copy all files and folders from your Windows installation ISO or DVD directly onto the root of the newly prepared USB drive..
