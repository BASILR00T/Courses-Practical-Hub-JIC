Disk Management Commands

General CMD Commands

These commands are run directly in the Command Prompt, not inside the `diskpart` utility.

format

Prepares a storage drive for use by creating a new file system. Warning: This erases all data on the specified drive.

C:\>format D: /fs:ntfs /q /v:Data
The type of the file system is RAW.
The new file system is NTFS.
QuickFormatting 953869M
Creating file system structures.
Format complete.

vol

Displays the volume label and serial number for a specific drive.

C:\>vol C:
 Volume in drive C is Windows
 Volume Serial Number is 1A2B-3C4D

label

Creates, changes, or deletes the volume label (name) of a disk.

C:\>label D: Backup

Diskpart - General Disk Management

A powerful utility for advanced disk management. Start it by typing diskpart in an elevated Command Prompt.

List & Select Commands

Used to view and select disks, partitions, or volumes to work on.

DISKPART>list disk

  Disk ###  Status         Size     Free     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
  Disk 0    Online          465 GB      0 B        *
  Disk 1    Online           58 GB    58 GB
DISKPART>select disk 1
Disk 1 is now the selected disk.
DISKPART>list partition
There are no partitions on this disk to show.

Create & Format Partitions

Commands to create, format, and assign letters to new partitions.

! Create a 20GB Primary Partition
DISKPART>create partition primary size=20000
DiskPart succeeded in creating the specified partition.

! Format the newly created partition (it is selected automatically)
DISKPART>format fs=ntfs label="Primary" quick
  100 percent completed
DiskPart successfully formatted the volume.

! Assign a drive letter
DISKPART>assign letter=G
DiskPart successfully assigned the drive letter or mount point.

Modify & Delete Partitions

Commands to change or remove existing partitions.

! Extend the selected volume by 5GB (5000MB)
DISKPART>extend size=5000
DiskPart successfully extended the volume.

! Shrink the selected volume by 10GB (10000MB)
DISKPART>shrink desired=10000
DiskPart successfully shrunk the volume by: 10 GB

! Remove the drive letter G from the selected volume
DISKPART>remove letter=G
DiskPart successfully removed the drive letter or mount point.

! Delete the currently selected partition (erases data)
DISKPART>delete partition
DiskPart successfully deleted the selected partition.

Information Commands

Commands to get details about disks and supported file systems.

! Get detailed information about the currently selected disk
DISKPART>detail disk
SAMSUNG MZ77Q1T0B/003 (SSD)
    Disk ID: {1234ABCD-5678-90EF-1234-567890ABCDEF}
    ...
    Volume ###  Ltr  Label        Fs     Type        Size     Status     Info
    ----------  ---  -----------  -----  ----------  -------  ---------  --------
    Volume 1         SYSTEM       FAT32  Partition    100 MB  Healthy    System
    Volume 2     C   Windows      NTFS   Partition    465 GB  Healthy    Boot

! See all supported file systems for formatting
DISKPART>filesystems

  File System  Type        Size
  -----------  ----------  -----------------------
  FAT          FAT         Up to 4 GB
  FAT32        FAT32       Up to 32 GB
  exFAT        exFAT       Up to 128 PB
  NTFS         NTFS        Up to 256 TB

Diskpart - VHD Management

Commands specifically for creating and managing Virtual Hard Disk (`.vhd` or `.vhdx`) files.

create vdisk

Creates a virtual hard disk file.

DISKPART>create vdisk file="D:\vhd\test.vhd" maximum=10000
  100 percent completed
DiskPart successfully created the virtual disk file.

select vdisk

Selects the specified VHD to work with.

DISKPART>select vdisk file="D:\vhd\test.vhd"
DiskPart successfully selected the virtual disk file.

attach vdisk

Attaches the selected VHD, making it appear as a physical disk to Windows.

DISKPART>attach vdisk
  100 percent completed
DiskPart successfully attached the virtual disk file.

expand vdisk

Expands the maximum size of a VHD. The VHD must be detached first.

DISKPART>expand vdisk maximum=20000
  100 percent completed
DiskPart successfully expanded the virtual disk.

detach vdisk

Detaches the selected VHD, making it "disappear" from the system.

DISKPART>detach vdisk
DiskPart successfully detached the virtual disk file.

Creating a Bootable USB Drive

Use `diskpart` to prepare a USB flash drive for installing Windows.

DISKPART>list disk
DISKPART>select disk 2      ! Assuming disk 2 is the USB drive
DISKPART>clean
DISKPART>create partition primary
DISKPART>select partition 1
DISKPART>active
DISKPART>format fs=fat32 quick label="WinSetup"
DISKPART>assign letter=J
DISKPART>exit

After running these commands, copy the entire contents of the Windows installation media (DVD or mounted ISO) to the root of the USB drive.
