Disk Management Commands

General CMD Commands

These commands are run directly in the Command Prompt, not inside the `diskpart` utility.

format

Prepares a storage drive for use by creating a new file system. Warning: This erases all data on the specified drive.

C:\>format D: /fs:ntfs /q /v:Data
The type of the file system is RAW.
The new file system is NTFS.
QuickFormatting 953869M
Creating file system structures.
Format complete.

vol

Displays the volume label and serial number for a specific drive.

C:\>vol C:
 Volume in drive C is Windows
 Volume Serial Number is 1A2B-3C4D

label

Creates, changes, or deletes the volume label (name) of a disk.

C:\>label D: Backup
Volume in drive D is Data
Volume Serial Number is ABCD-EF98
Volume label (11 characters, ENTER for none)? Backup

Diskpart - General Disk Management

A powerful utility for advanced disk management. Start it by typing diskpart in an elevated Command Prompt.

List & Select Commands

Used to view and select disks, partitions, or volumes to work on.

DISKPART>list disk
  Disk ###  Status         Size     Free     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
  Disk 0    Online          931 GB      0 B        *
  Disk 1    Online           58 GB    58 GB
DISKPART>select disk 1
Disk 1 is now the selected disk.
DISKPART>list partition
  Partition ###  Type              Size     Offset
  -------------  ----------------  -------  -------
  Partition 1    Primary             20 GB  1024 KB
  Partition 2    Extended            38 GB    20 GB
  Partition 3    Logical             10 GB    20 GB

Create & Format Partitions

This example shows how to partition a new disk using the modern GPT style.

! Select, clean, and convert the disk to GPT
DISKPART> select disk 1
DISKPART> clean
DISKPART> convert gpt
! Create a 100GB primary partition
DISKPART> create partition primary size=100000
DiskPart succeeded in creating the specified partition.
! Format and assign a letter
DISKPART> format fs=ntfs quick label="Windows"
DiskPart successfully formatted the volume.
DISKPART> assign letter=W
DiskPart successfully assigned the drive letter or mount point.

Modify & Delete Partitions

Commands to change or remove existing partitions.

! Extend the selected volume by 5GB (5000MB)
DISKPART>extend size=5000
DiskPart successfully extended the volume.

! Shrink the selected volume by 10GB (10000MB)
DISKPART>shrink desired=10000
DiskPart successfully shrunk the volume by: 10 GB

! Remove the drive letter G from the selected volume
DISKPART>remove letter=G
DiskPart successfully removed the drive letter or mount point.

! Delete the currently selected partition (erases data)
DISKPART>delete partition
DiskPart successfully deleted the selected partition.

Information Commands

Commands to get details about disks and supported file systems.

! Get detailed information about the currently selected disk
DISKPART>detail disk
SAMSUNG MZ77Q1T0B/003 (SSD)
    Disk ID: {1234ABCD-5678-90EF-1234-567890ABCDEF}
    ...

! See all supported file systems for formatting
DISKPART>filesystems

  File System  Type        Size
  -----------  ----------  -----------------------
  NTFS         NTFS        Up to 256 TB
  ...

Diskpart - VHD Management & MBR Partitioning

This workflow shows the complete process of creating a VHD, converting it to the legacy MBR style, and creating primary, extended, and logical partitions.

! Create a 5GB VHD file
DISKPART>create vdisk file="D:\vhd\mbr_disk.vhd" maximum=5000
  100 percent completed
DiskPart successfully created the virtual disk file.

! Select and attach the VHD to the system
DISKPART>select vdisk file="D:\vhd\mbr_disk.vhd"
DiskPart successfully selected the virtual disk file.
DISKPART>attach vdisk
  100 percent completed
DiskPart successfully attached the virtual disk file.

! Convert the attached VHD to MBR style
DISKPART>convert mbr
DiskPart successfully converted the selected disk to MBR format.

! Create a 2GB Primary Partition (Max 4 on MBR)
DISKPART>create partition primary size=2000
DiskPart succeeded in creating the specified partition.

! Create an Extended Partition with the remaining space (Max 1 per disk)
DISKPART>create partition extended
DiskPart succeeded in creating the specified partition.

! Create a 1GB Logical drive inside the extended partition (Unlimited logical drives)
DISKPART>create partition logical size=1000
DiskPart succeeded in creating the specified partition.

! Select the first partition (the primary one) to format it
DISKPART>select partition 1
Partition 1 is now the selected partition.

! Format and assign a letter
DISKPART>format fs=ntfs quick label="VHD_Primary"
  100 percent completed
DiskPart successfully formatted the volume.
DISKPART>assign letter=V
DiskPart successfully assigned the drive letter or mount point.

Creating a Bootable USB Drive

Use Diskpart to prepare a USB flash drive to be a bootable Windows installation medium. Warning: This will erase all data on the USB drive.

// 1. Identify your USB drive (e.g., Disk 2)
DISKPART> list disk
  Disk ###  Status         Size     Free     Dyn  Gpt
  --------  -------------  -------  -------  ---  ---
  Disk 0    Online          465 GB      0 B        *
  Disk 1    Online          931 GB      0 B        *
  Disk 2    Online           58 GB      0 B

// 2. Select the USB drive
DISKPART> select disk 2
Disk 2 is now the selected disk.

// 3. Wipe the drive
DISKPART> clean
DiskPart succeeded in cleaning the disk.

// 4. Create a primary partition
DISKPART> create partition primary
DiskPart succeeded in creating the specified partition.

// 5. Select and mark the partition as active (bootable)
DISKPART> select partition 1
Partition 1 is now the selected partition.
DISKPART> active
DiskPart marked the current partition as active.

// 6. Format the partition as FAT32
DISKPART> format fs=fat32 quick label="WinSetup"
  100 percent completed
DiskPart successfully formatted the volume.

// 7. Assign a drive letter
DISKPART> assign letter=W
DiskPart successfully assigned the drive letter or mount point.

// 8. Exit diskpart
DISKPART> exit
Leaving DiskPart...

Warning: After completing these steps, you must copy all files and folders from your Windows installation ISO or DVD directly onto the root of the newly prepared USB drive..
