Device Manager

Accessing and Using Device Manager

Device Manager is a vital tool for managing hardware components and their drivers. It can be accessed quickly by running the command devmgmt.msc.

To open Device Manager: Press Windows Logo + R, type devmgmt.msc, and press Enter.

Key Functions:

  1. View Devices: See all hardware installed on the computer, organized by category. A yellow exclamation mark indicates a problem with the device or its driver.
  2. Update Driver: Right-click a device and select "Update driver" to search for a new driver automatically or manually.
  3. Roll Back Driver: If a new driver causes issues, you can revert to the previously installed one. Right-click the device -> Properties -> Driver tab -> Roll Back Driver.
  4. Disable/Enable Device: Temporarily turn a device off or on without uninstalling it.
  5. Uninstall Device: Completely remove a device and its driver from the system.
