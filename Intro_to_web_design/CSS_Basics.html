CSS Basics

Learn how to style your web pages with CSS.

The Style

CSS (Cascading Style Sheets) controls how your HTML looks. It handles colors, fonts, layout, and spacing.

1. Syntax

CSS works by selecting an element and applying styles to it:

selector {
    property: value;
}

2. Simple Examples

/* Select by Tag Name */
p {
    color: blue;
    font-size: 16px;
}

/* Select by Class (starts with .) */
.my-button {
    background-color: red;
    border-radius: 5px;
}

/* Select by ID (starts with #) */
#main-header {
    text-align: center;
}

3. The Box Model

Every element in web design is a box. It consists of:

  • Content: The actual text or image.
  • Padding: Space inside the border (around content).
  • Border: The line around the padding.
  • Margin: Space outside the border (separates elements).
