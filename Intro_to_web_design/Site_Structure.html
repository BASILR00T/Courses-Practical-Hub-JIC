Designing a Multi-Page Site

How to structure and link multiple pages effectively.

Site Structure

A real website consists of multiple pages linked together. Here is how to structure it properly.

1. Folder Structure

Keep your project organized. Don't dump everything in one folder.

/MyWebsite
    index.html          (Your Home Page)
    about.html          (About Page)
    contact.html        (Contact Page)
    /css
        style.css       (One style file for ALL pages)
    /images
        logo.png
        banner.jpg

2. Linking Pages

On your index.html, you link to other pages using relative paths:

<nav>
    <a href="index.html">Home</a>
    <a href="about.html">About</a>
    <a href="contact.html">Contact</a>
</nav>

3. Sub-pages (e.g., about.html)

When creating a sub-page, copy the basic structure from your index.html so the header, footer, and links remain consistent. Link to the same CSS file to maintain the design.

<head>
    ...
    <!-- Use ../ if the css folder is one level up -->
    <link rel="stylesheet" href="css/style.css">
</head>
