DHCP Server Setup (Win Server 2022)

A complete guide to configuring DHCP & DNS on Windows Server using Hyper-V.

1. Create Virtual Machine in Hyper-V

Setting up the environment for the server.

  1. Open Hyper-V Manager.
  2. Click New > Virtual Machine.
  3. Name: "DHCP_[Last 3 Digits of ID]"
    (Example: DHCP_789).
  4. Generation: Select Generation 2 (Recommended).
  5. Memory: Assign at least 4096 MB (4GB).
  6. Networking: Select your "External Switch".
  7. Installation: Select the Windows Server 2022 ISO.

2. Install DHCP & DNS Roles

Important: Assign a Static IP Address (e.g., 192.168.1.10) before starting.
  1. Open Server Manager > Add Roles and Features.
  2. Check DHCP Server and DNS Server.
  3. Click Install.
  4. After install, click the Notification flag and select "Complete DHCP Configuration".

3. Create a DHCP Scope

Defining the pool of IP addresses with specific local network requirements.

  1. Open Tools > DHCP.
  2. Right-click IPv4 > New Scope.
  3. IP Address Range:
    • Start IP: 192.168.1.3
    • End IP: 192.168.1.254
    • Subnet Mask: 255.255.255.0
  4. Add Exclusions (Execute):
    • Start IP: 192.168.1.10
    • End IP: 192.168.1.30
    • (These addresses will NOT be distributed)
  5. Lease Duration: Set "Limited to" days: 100 Days.
  6. Configure DHCP Options: Select "Yes, I want to configure these options now".
  7. Router (Default Gateway): SKIP THIS STEP. Leave it blank (Local Network Only).
  8. DNS Server: Ensure the server's own IP is listed (e.g., 192.168.1.10).
  9. Activate the Scope.

4. Join Server to Domain

Joining the server to an existing Active Directory.

  1. Open System Properties.
  2. Click Change... settings.
  3. Select Domain and type the name (e.g., mycompany.local).
  4. Enter Domain Admin credentials.
  5. Restart the server.

5. Client: Obtain IP from DHCP

Verify on a Windows 10/11 Client Machine.

  1. Set Network Adapter to "Obtain an IP address automatically".
  2. Run these commands in CMD:
C:\> ipconfig /release
C:\> ipconfig /renew
C:\> ipconfig /all
