Network Operating Systems (Linux)

1. File System Navigation & Management

Basic commands to navigate directories and manage files.

Navigation (pwd, cd, ls)

[root@comp1 ~]# pwd
/root # Displays current working directory

[root@comp1 ~]# ls -l   # Long listing (permissions, owner, size)
[root@comp1 ~]# ls -a   # Show hidden files (starts with .)
[root@comp1 ~]# ls -R   # Recursive listing (tree structure)

[root@comp1 ~]# cd /home  # Change directory to /home
[root@comp1 ~]# cd ..     # Go up one level
[root@comp1 ~]# cd ~      # Go to logged-in user's home directory

File Creation & Manipulation

# Create empty files or update timestamp
[root@comp1 ~]# touch file1.txt file2.txt

# View file contents
[root@comp1 ~]# cat file1.txt
[root@comp1 ~]# cat -n file1.txt # View with line numbers
[root@comp1 ~]# head file1.txt   # First 10 lines
[root@comp1 ~]# tail file1.txt   # Last 10 lines

# Create Directories
[root@comp1 ~]# mkdir mydir
[root@comp1 ~]# mkdir -p dir1/dir2/dir3 # Create nested directories

Copy, Move, Remove

[root@comp1 ~]# cp source.txt dest.txt      # Copy file
[root@comp1 ~]# cp -r folder1 folder2       # Copy directory recursively

[root@comp1 ~]# mv file1.txt newname.txt    # Rename file
[root@comp1 ~]# mv file1.txt /tmp/          # Move file

[root@comp1 ~]# rm file1.txt                # Remove file
[root@comp1 ~]# rm -rf mydir                # Remove directory forcefully

2. VI Editor Commands

The Visual Editor has two main modes: Command Mode (default) and Input Mode.

To open a file: vi filename

Switching Modes

  • i : Insert mode (before cursor)
  • a : Append mode (after cursor)
  • o : Open new line below
  • Esc : Return to Command Mode

Navigation (Command Mode)

  • h Left | j Down | k Up | l Right
  • w : Next word | b : Previous word
  • $ : End of line | 0 : Start of line
  • G : End of file | 1G : Start of file

Editing & Deleting

  • x : Delete character
  • dd : Delete current line (Cut)
  • yy : Copy current line (Yank)
  • p : Paste below cursor
  • u : Undo last change

Saving & Exiting

:w   # Save
:wq  # Save and Quit (or ZZ)
:q!  # Quit without saving (Discard changes)

3. Permissions & Ownership

Managing file access levels for User (u), Group (g), and Others (o).

chmod (Change Mode)

# Symbolic Method
[root@comp1 ~]# chmod u+x script.sh  # Add execute permission for User
[root@comp1 ~]# chmod g-w file.txt   # Remove write permission for Group
[root@comp1 ~]# chmod a=r file.txt   # Set read-only for All

# Numeric Method (r=4, w=2, x=1)
[root@comp1 ~]# chmod 755 script.sh
# 7 (rwx) for User
# 5 (r-x) for Group
# 5 (r-x) for Others

Ownership (chown & chgrp)

[root@comp1 ~]# chown ahmed file.txt        # Change user owner
[root@comp1 ~]# chgrp students file.txt     # Change group owner
[root@comp1 ~]# chown -R ahmed /var/www     # Recursive change
Default Permissions (umask 022):
Files: 644 (rw- r-- r--)
Directories: 755 (rwx r-x r-x)

4. System Power Commands

# Shutdown
[root@comp1 ~]# shutdown -P now   # Power off immediately
[root@comp1 ~]# shutdown -r now   # Reboot immediately
[root@comp1 ~]# shutdown -c       # Cancel scheduled shutdown

# Other aliases
[root@comp1 ~]# reboot
[root@comp1 ~]# poweroff
[root@comp1 ~]# halt              # Halts system (does not power off)
