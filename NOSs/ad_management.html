Manage OUs, Users, and Groups

Organize your network resources and manage user access through Active Directory.

  1. On your "DC-Server", open Active Directory Users and Computers from the Tools menu in Server Manager.
  2. Create an OU (Organizational Unit): Right-click your domain -> New -> Organizational Unit. (Ous -> Jubail , Khobar , Riyadh).
  3. Create a Group: Right-click on the "Ous" -> New -> Group. ("Sales" , "Marketing").
  4. Create a User: Right-click on the "Ous -> Jubail , Khobar , Riyadh" -> New -> User. Fill in the details and set a password.
  5. Add User to Group: Douple-click on the Groups ("Sales" , "Marketing") -> "Member". Enter the username ("Rami , Jamal ...") and click Check nems -> apply.

If there are any mistakes or wrong info please feel free to contact me on BSA717@outlook.sa.
